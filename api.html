How to register a wallet
To register a Tiros wallet you would need to send an HTTP POST request to http://miceganich.is-great.org/regg.php
Post request must include "name" and "pass" values.
Example shown below in javascript: 
<form action="http://miceganich.is-great.org/regg.php" method="post">
  <div>
    <input name="name" value="">
  </div>
  <div>
    <input name="pass" value="">
  </div>
  <div>
    <button>Send</button>
  </div>
</form>
How to login check a wallet
To check user credintials you would need to send an HTTP POST request to http://miceganich.is-great.org/Login.php
Example shown below in javascript:
<form action="http://miceganich.is-great.org/Login.php" method="post">
  <div>
    <input name="name" value="">
  </div>
  <div>
    <input name="pass" value="">
  </div>
  <div>
    <button>Send</button>
  </div>
</form>

if succeed, should return "true, wallet: (wallet number, no brackets)"
else - "wrong login credentials" How to check a currency on the wallet
To check currency on any wallet registered, you would need to send an HTTP POST request to http://miceganich.is-great.org/getcount.php
Example shown below in javascript:
<form action="http://miceganich.is-great.org/getcount.php" method="post">
  <div>
    <input name="wallet" value="">
  </div>
  <div>
    <button>Send</button>
  </div>
</form>
How to transfer a currency to an another wallet
To transfer currency to any wallet registered, you would need to send an HTTP POST request to http://miceganich.is-great.org/transferr.php
if some credentials are wrong returns "wrong login credentials" or "not enough Torus currency coins", else "New records created successfully".
Example shown below in javascript:
<form action="http://miceganich.is-great.org/transferr.php" method="post">
  <div>
   
    <input name="name" value="">
  </div>
  <div>
   
    <input name="pass" value="">
  </div>
    <div>
   
    <input name="to" value="">
  </div>
    <div>
    
    <input name="count" value="">
  </div>
  <div>
    <button>Send</button>
  </div>
</form>

There's no matter which language you use to send the request, javascript is just the most easiest example, you would be better using php's curl for these...
